Matching Duolingo's Layout

Featured card at top — dark gradient, bold title, illustration, CTA button. Like Duolingo's "Today's Review" card.

"Your collections" below — dark bordered cards with icon + title + description + illustration. Matches Duolingo's Mistakes/Words/Stories layout.

Dark theme — Duolingo's practice hub uses their dark navy background. We match that.

Emoji placeholders — in production, these would be real illustrated icons like Duolingo's microphone/headphones/cards.

Three Practice Modes

Smart Practice (featured) — the main mode. Algorithm picks 10 questions, wrong answers recycle. Always the hero card.

Mistakes — re-practice questions you got wrong in past sessions. Badge shows count of unresolved mistakes. Uses existing mistake tracking.

Daily Challenge — existing /practice/daily, 5 themed questions per day. Already built.

Review — existing /practice/review (spaced repetition). Already built.
