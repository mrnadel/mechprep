How Names Were Scored

Pronounceability (25%)

Can you say it on a podcast? Tell a friend in a noisy bar? Spell it after hearing it once?

Mascot / Brand Potential (25%)

Does it suggest a visual identity? Can you draw a logo from the name alone? Think Duolingo's owl.

Emotional Resonance (20%)

Does it make you smile? Feel encouraged? Want to use the app? Positive vibes sell.

Memorability (15%)

Would you remember it tomorrow? Weird/unexpected combos stick: "studtaco" beats "learnplatform".

Social Media Virality (10%)

Is it hashtag-friendly? Would people share it? Does it work as @handle on all platforms?

Length & Simplicity (5%)

Shorter is better for logos, URLs, and word-of-mouth. Sweet spot: 7-9 characters.
